# Constitución del Proyecto: Web Grupo iTTel

## 1. Identidad y Tono
* **Estilo Visual:** Brutalismo corporativo suave, estilo "American Tower" o consultoras multinacionales. Altamente profesional, técnico y minimalista.
* **Colores:** Fondo blanco (#FFFFFF) y grises claros (#F8FAFC, #F1F5F9). Textos casi negros (#0F172A) y grises medios (#475569). Acento: Azul corporativo (#227DB3).
* **Tono de voz:** Directo, basado en datos, cero texto de relleno ("sarasa"). B2B puro.

## 2. Reglas Técnicas (No negociables)
* **Stack:** HTML5 semántico, CSS3 puro (sin frameworks como Tailwind o Bootstrap), JavaScript Vanilla.
* **Componentes visuales:** * PROHIBIDO el uso de íconos decorativos (ej. FontAwesome) en descripciones de servicios.
  * Los bordes deben ser sutiles (1px solid #E2E8F0) con un radio mínimo (4px a 8px).
  * Sombras mínimas o nulas (diseño flat/editorial).
* **Imágenes:** Si falta una imagen, dejar el placeholder explícito: ``.
